# Fract Demo

## Building the demo

```sh
# Navigate to this directory in terminal
cd ./demo

# Install dependencies
npm i

# Build TypeScript and bundle JavaScript
npm run buildDemo
```

## Running the demo

```sh
# Start static web-server
npx static -p 8000 
```

Then, open the demo in your browser:
[http://localhost/8000](http://localhost/8000)
